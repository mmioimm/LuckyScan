# LuckyScan
script for port scan

> 学习go而写的端口扫描工具

- 基于yml，可在config.yml修改不同服务的扫描的默认端口号及用户名密码字典
- 在dic文件中，可针对性的添加自己的字典，也可使用默认提供的

![image](https://user-images.githubusercontent.com/19258899/147410465-78949e06-3ada-40b4-a469-542e60929f6b.png)


usage: 

        -h 参数指定ip地址段
        
        -s 参数指定要扫描的服务名称
